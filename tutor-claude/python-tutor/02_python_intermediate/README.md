# Phase 2: Python 중급 개념

## 학습 목표
Python의 강력한 기능들을 활용하여 더 우아하고 효율적인 코드를 작성합니다.

## 학습 순서

### 1. 데코레이터 (01_decorators.py)
**학습 시간**: 3-4시간

**핵심 개념**:
- 함수 데코레이터의 작동 원리
- 클래스 데코레이터
- 매개변수가 있는 데코레이터
- 여러 데코레이터 중첩
- 실무 활용 패턴 (로깅, 캐싱, 인증 등)

### 2. 제너레이터와 이터레이터 (02_generators_iterators.py)
**학습 시간**: 3-4시간

**핵심 개념**:
- 이터레이터 프로토콜
- 제너레이터 함수와 표현식
- yield와 yield from
- 메모리 효율적인 데이터 처리
- 무한 시퀀스

### 3. 컨텍스트 매니저 (03_context_managers.py)
**학습 시간**: 2-3시간

**핵심 개념**:
- with 문의 작동 원리
- __enter__와 __exit__ 구현
- contextlib 모듈 활용
- 리소스 관리 패턴

### 4. 컴프리헨션 심화 (04_comprehensions.py)
**학습 시간**: 2-3시간

**핵심 개념**:
- 리스트/딕셔너리/집합 컴프리헨션
- 중첩 컴프리헨션
- 조건부 표현식
- 성능 최적화

### 5. 람다와 함수형 프로그래밍 (05_lambda_map_filter.py)
**학습 시간**: 3-4시간

**핵심 개념**:
- 람다 함수의 적절한 사용
- map, filter, reduce 활용
- functools 모듈
- 함수형 프로그래밍 패턴
