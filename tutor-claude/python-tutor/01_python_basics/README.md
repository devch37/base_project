# Phase 1: Python 기초 다지기

## 학습 목표
Python의 기본 문법과 핵심 개념을 완벽하게 이해하고, 객체지향 프로그래밍의 기초를 다집니다.

## 학습 순서

### 1. 변수와 자료형 (01_variables_and_types.py)
**학습 시간**: 2-3시간

**핵심 개념**:
- Python의 동적 타이핑 시스템 이해
- 기본 자료형: int, float, str, bool
- 컬렉션 자료형: list, tuple, dict, set
- 타입 변환과 타입 체크
- 변수 네이밍 컨벤션

**실습 목표**:
- 다양한 자료형을 활용한 데이터 표현
- 적절한 자료구조 선택 능력 배양

### 2. 제어문 (02_control_flow.py)
**학습 시간**: 2-3시간

**핵심 개념**:
- 조건문: if, elif, else
- 반복문: for, while
- 제어 키워드: break, continue, pass
- 예외 처리: try, except, finally
- 컨텍스트 매니저: with 문

**실습 목표**:
- 복잡한 로직을 조건문과 반복문으로 표현
- 예외 상황을 안전하게 처리하는 코드 작성

### 3. 함수 (03_functions.py)
**학습 시간**: 3-4시간

**핵심 개념**:
- 함수 정의와 호출
- 매개변수와 인자 (위치 인자, 키워드 인자, 기본값)
- *args와 **kwargs
- 람다 함수
- 함수 스코프와 클로저
- 일급 객체로서의 함수

**실습 목표**:
- 재사용 가능한 함수 작성
- 함수형 프로그래밍의 기초 이해

### 4. 클래스와 객체지향 프로그래밍 (04_classes_and_oop.py)
**학습 시간**: 4-5시간

**핵심 개념**:
- 클래스와 객체의 개념
- 생성자와 소멸자
- 인스턴스 변수와 클래스 변수
- 메서드 종류: 인스턴스 메서드, 클래스 메서드, 스태틱 메서드
- 캡슐화와 정보 은닉
- 상속과 다형성
- 매직 메서드 (던더 메서드)

**실습 목표**:
- 객체지향 설계의 기초 이해
- 상속과 다형성을 활용한 코드 재사용

### 5. 모듈과 패키지 (05_modules_and_packages.py)
**학습 시간**: 2-3시간

**핵심 개념**:
- 모듈의 개념과 import 방법
- 패키지 구조
- __init__.py의 역할
- 상대 경로와 절대 경로 import
- 표준 라이브러리 활용

**실습 목표**:
- 코드를 모듈화하여 관리
- 재사용 가능한 라이브러리 구조 이해

### 6. 고급 기능 (06_advanced_features.py)
**학습 시간**: 3-4시간

**핵심 개념**:
- 리스트 컴프리헨션
- 딕셔너리 컴프리헨션
- 제너레이터 표현식
- 데코레이터 기초
- 파일 입출력
- 정규 표현식

**실습 목표**:
- Python의 강력한 기능을 활용한 간결한 코드 작성
- 파일 처리와 텍스트 파싱 능력 배양

## 실습 방법

1. **순차적으로 학습**: 파일 번호 순서대로 진행하세요.
2. **코드 실행**: 각 파일을 실행하면서 결과를 확인하세요.
   ```bash
   python 01_variables_and_types.py
   ```
3. **주석 정독**: 코드의 '어떻게'뿐만 아니라 '왜'를 설명하는 주석을 꼼꼼히 읽으세요.
4. **실험**: 코드를 변경해보고 어떻게 동작하는지 관찰하세요.
5. **연습 문제**: 각 파일 하단의 연습 문제를 풀어보세요.
