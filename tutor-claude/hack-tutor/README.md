# Hack Tutor (Java/Spring)

윤리적/방어적 보안(화이트햇) 관점의 학습 자료입니다. **실습은 반드시 로컬/허가받은 환경에서만** 진행하세요.

## 구성
- `chapters/` 단계별 이론과 체크리스트
- `labs/` 로컬 실습 가이드 (Java/Spring 중심)
- `resources/` 참고 자료와 템플릿
- `app/` 로컬 실습용 Spring Boot 앱

## 학습 방법
1. 각 챕터의 **목표 → 핵심 개념 → 체크리스트** 순서로 읽기
2. 해당하는 `labs/`를 실행해 “방어 관점”에서 직접 적용
3. 본인 프로젝트에 체크리스트를 적용해 리팩터링

## 과제/정답
- `labs/assignments/` 과제
- `labs/solutions/` 정답/설명

## 로컬 실습 환경 권장
- JDK 17+
- Spring Boot 3.x
- Gradle 또는 Maven
- Docker (선택)

## 안전 원칙
- 실습은 로컬/CTF/허가받은 환경에서만 수행
- 실제 서비스, 타인 소유 시스템에 대한 테스트 금지
- 로그/데이터는 민감정보 마스킹
