# Lab 0 (Assignment). 로컬 실습 환경 만들기

## 목표
- 로컬 전용 Spring Boot 앱을 만든다.
- 이후 실습의 기반을 준비한다.

## 과제
- Spring Boot 프로젝트를 생성한다.
- `GET /ping` 엔드포인트를 만든다.
- 로컬에서 애플리케이션을 실행한다.

## 힌트
- 의존성: `Spring Web`, `Spring Security`, `Validation`
- Java 17 이상

## 체크리스트
- 외부 서비스/DB 없이 로컬에서 동작하는가
- 테스트 데이터가 민감정보를 포함하지 않는가
