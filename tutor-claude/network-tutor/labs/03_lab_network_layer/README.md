# Lab 03 — IP, 서브네팅, 라우팅, NAT

## 실습 목표
- 서브네팅 계산을 코드로 자동화하고 원리를 확인한다
- traceroute로 실제 라우팅 경로를 분석한다
- NAT 동작을 직접 관찰한다

## 파일 목록
| 파일 | 설명 |
|------|------|
| `step1_subnet_calculator.py` | 서브네팅 계산 실습 & 시각화 |
| `step2_routing_trace.sh` | traceroute 분석 & TTL 실험 |
| `step3_nat_observer.sh` | NAT 관찰 (공인 IP vs 사설 IP) |
