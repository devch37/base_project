# Lab 02 — ARP & MAC 주소 직접 확인

## 실습 목표
- ARP Request/Reply 패킷을 캡처하고 MAC 주소 해석 과정을 확인한다
- ARP 캐시 조작으로 Gratuitous ARP 동작을 이해한다
- VLAN 개념을 Docker 네트워크 격리로 체험한다

## 파일 목록
| 파일 | 설명 |
|------|------|
| `step1_arp_observe.sh` | ARP 캡처 & 캐시 실습 |
| `step2_vlan_docker.sh` | Docker로 VLAN 격리 체험 |
| `step3_arp_analyzer.py` | ARP 동작 Python 시뮬레이터 |
