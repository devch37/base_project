# Lab 07 — 네트워크 보안 실습

## 실습 목표
- macOS pf/iptables로 방화벽 규칙을 직접 작성하고 테스트한다
- nmap으로 포트 스캔 결과를 분석하고 취약점을 파악한다
- Rate Limiting을 Python으로 직접 구현한다

## 파일 목록
| 파일 | 설명 |
|------|------|
| `step1_port_scan.sh` | nmap 포트 스캔 & 서비스 감지 |
| `step2_firewall_lab.sh` | 방화벽 규칙 실습 |
| `step3_synflood_sim.py` | SYN Flood 시뮬레이션 & 방어 확인 |
