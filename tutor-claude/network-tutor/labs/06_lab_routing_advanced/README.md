# Lab 06 — 라우팅 심화 & 클라우드 VPC 시뮬레이션

## 실습 목표
- Docker 네트워크로 멀티-서브넷 라우팅을 직접 구성한다
- 정적 라우팅과 동적 라우팅의 차이를 체험한다
- BGP Anycast 개념을 DNS로 직접 확인한다

## 파일 목록
| 파일 | 설명 |
|------|------|
| `step1_multi_subnet.sh` | Docker로 멀티 서브넷 라우팅 구성 |
| `step2_bgp_anycast.sh` | BGP/Anycast 개념 실습 |
| `step3_vpc_simulator.py` | VPC 라우팅 테이블 시뮬레이터 |
